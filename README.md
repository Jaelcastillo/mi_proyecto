# Mi Proyecto en Python con Poetry

##  Descripción
Este proyecto es una aplicación simple en Python que solicita el nombre del usuario y realiza una conexión a una API utilizando la librería `requests`. Si la conexión es exitosa, muestra un mensaje en pantalla.

##  Tecnologías utilizadas
- Python 3.14
- Poetry
- Librería requests

##  Instalación
1. Clonar el repositorio: git clone https://github.com/Jaelcastillo/mi_proyecto.git
2.  Entrar al proyecto:cd mi_proyecto
3.  Instalar dependencias: poetry install
4.  # Ejecución------ con poetry run python main.py
